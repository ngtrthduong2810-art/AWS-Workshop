### Outcomes of Week 8:

#### Knowledge

**AWS Identity and Access Management (IAM)**

- Understand the purpose of AWS IAM in managing authentication and authorization.
- Learn how IAM Users, Groups, Roles, and Policies work together.
- Understand the principle of least privilege when assigning permissions.
- Learn the differences between AWS managed policies and customer managed policies.
- Understand IAM credential management and access control.

**AWS Security**

- Learn how Multi-Factor Authentication (MFA) improves account security.
- Understand password policies and credential protection.
- Learn AWS identity security best practices.
- Understand the importance of securing root account credentials.

**Amazon Virtual Private Cloud (Amazon VPC)**

- Understand the architecture of Amazon VPC.
- Learn the concepts of CIDR blocks, Subnets, and IP addressing.
- Understand the purpose of Internet Gateways and Route Tables.
- Learn the differences between Public and Private Subnets.

**Network Security**

- Understand how Security Groups protect AWS resources.
- Learn the purpose of Network Access Control Lists (Network ACLs).
- Configure inbound and outbound network traffic rules.
- Understand secure network design principles in AWS.
